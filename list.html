List's in HTML

There are mainly 3 types of lists in HTML, they are :

1. Ordered List
An ordered list typically is a numbered list of items. HTML 3.0 gives you the ability to control the sequence number - to continue where the previous list left off, or to start at a particular number.For example
  1. HTML
  2. CSS
  3. Javascript
  4. NodeJS
2. Unordered List
An unordered list typically is a bulleted list of items. HTML 3.0 gives you the ability to customise the bullets, to do without bullets and to wrap list items horizontally or vertically for multicolumn lists.For example
  • Devi
  • Surya
  • Maha
  • Achuu
3. Description List
Description lists are used to associate terms with their relevant descriptions and branch learning is important. This is done by utilizing the description list element.
